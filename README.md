# 小型网站管理系统
## 本项目主要基于django框架开发的全栈项目，前端为css+html，后端为全Python实现
## 环境管理使用的是anaconda,数据库使用mysql数据库，前后端数据传输采用orm技术
## 本项目目的是实现一个小型网站管理系统,实现一个优质网站汇总
## 可自行添加网址以及备注内容 
## 当前项目已初步完成需求内容 
## 该项目已于2024.12月上旬完成 上传完整代码 数据库文件 以及 环境依赖
## 欢迎各位对项目内容提出改进方向和优化建议
## 警告 本项目仅用于学习练习 不得用于商用
